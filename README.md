# VinuHub Staking

A decentralized staking platform for VIN tokens on VinuChain, built with Next.js and TypeScript.

## Features
- Connect MetaMask wallet (auto/manual) on VinuChain (Chain ID: 207).
- Stake VIN tokens (min 10 VIN, 7-day lock, ~15% APY).
- View dashboard (staked amount, lock time, rewards).
- Unstake or emergency unstake (10% fee).
- Leaderboard of top stakers.
- Responsive UI with animations (cubic-bezier, radial gradient buttons).
